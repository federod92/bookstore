#Challenge
El challenge corresponde a GENUX

### Objective

Your assignment is to implement a bookstore REST API using PHP and no framework.

### Brief

Lohgarra, a Wookie from Kashyyyk, has a great idea. She wants to build a marketplace that allows her and her friends to
self-publish their adventures and sell them online to other Wookies. The profits would then be collected and donated to purchase
medical supplies for an impoverished Ewok settlement.

### Tasks

-   Implement assignment using:
    -   Language: **PHP**
    -   Framework: **no framework**
-   Implement a REST API returning JSON or XML based on the `Content-Type` header
-   Implement a book model. Each book should have a title, description, author (your custom user model), cover image and price
    -   Choose the data type for each field that makes the most sense
-   Implement REST endpoints for the `/books` resource
    -   No authentication required
    -   Allows only GET (List/Detail) operations
-   Upload the code to public GIT repository 
    

### Evaluation Criteria

-   **PHP** best practices
-   No framework

### CodeSubmit

Please organize, design and document your code as if it were
going into production - then push your changes to the master branch.

All the best and happy coding,


#Qué hace el proyecto
Implementa una Api Rest (sina utenticación), por lo cual no es una API REST real

#Por qué el proyecto es útil
Es útil para aprendizaje. No para implementarla ya que no es una API REST real.

#Cómo pueden comenzar los usuarios con el proyecto.

1) Instalar un XAMPP o similar, que cuente servicios Mysql y Apache.
2) Descargar el proyecto y copiar en la carpeta raiz, en este caso C:\xampp\htdocs\bookstore
3) Lanzar los servicos de MySql y Apache
4) Abrir el un navegador cual quiera e ingresar al siguiente link http://localhost/bookstore/
5) El archivo index.php devuelve un JSON con todos los libros cargados en la base de datos
